- Diagrams (via plantuml or graphviz via extensions)

- Include directives

[source,python]
----
def fibonacci(n):
    """Return the nth Fibonacci number."""
    a, b = 0, 1
    for _ in range(n):
        yield a
        a, b = b, a + b
----
